# FlipClock.js

A full featured, themeable, type safe, and well tested library for clocks, timers, counters, and flipboards.

## Install

```bash
pnpm add flipclock
```

```bash
npm i flipclock
```

```bash
yarn add flipclock
```

## Documentation

To check out docs, visit [https://flipclockjs.com/](https://flipclockjs.com/).
